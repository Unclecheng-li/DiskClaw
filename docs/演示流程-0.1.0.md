# 磁盘清理大虾 0.1.0 演示流程

## 演示目标

通过一条清晰的桌面端操作路径，展示磁盘清理大虾从“发现问题”到“执行清理”再到“恢复回退”的完整闭环。

## 演示数据

可先执行：

```bash
npm run demo:prepare
```

生成目录：

- [demo/demo-data](C:/Users/UncleC/Desktop/DiskClaw/demo/demo-data)

这套演示数据会包含：

- 重复安装镜像文件
- 大型压缩包
- 临时日志
- 构建缓存痕迹

## 建议演示顺序

### 1. 启动桌面端

- 启动 [DiskClaw-Setup-0.1.0.exe](C:/Users/UncleC/Desktop/DiskClaw/dist/DiskClaw-Setup-0.1.0.exe) 安装后的客户端，或直接运行 [win-unpacked](C:/Users/UncleC/Desktop/DiskClaw/dist/win-unpacked) 中的 `DiskClaw.exe`
- 展示应用图标、标题栏和“关于磁盘清理大虾”

### 2. 首启引导

- 展示首启引导浮层
- 点击“填充推荐值”或“开始使用”
- 切到“设置”页展示 LLM 与扫描参数

### 3. 配置模型

- 在 LLM 配置中填写示例模型地址
- 点击“测试连接”
- 点击“保存配置”

### 4. 发起扫描与方案生成

- 回到“总览”
- 设置扫描目标路径
- 点击“生成清理方案”
- 展示顶部任务状态条的进度、阶段、文件数与目录数

### 5. 展示首页摘要

- 展示可回收空间、候选项数量、重复文件组、热点目录
- 展示低 / 中 / 高风险数量
- 展示优先事项
- 点击优先事项，触发对应 Dry Run

### 6. 执行清理

- 切到“执行与报告”
- 分别展示：
  - 普通清理 Dry Run
  - 重复文件 Dry Run
  - 热点目录 Dry Run
- 展示清理结果统计卡片

### 7. 任务与报告回看

- 展示任务列表、筛选和详情
- 展示报告列表、筛选和详情
- 展示摘要视图 + 原始 JSON

### 8. 隔离区恢复

- 点击“查看隔离区”
- 展示隔离项列表
- 点击“恢复最近一项”或“恢复选中项”
- 展示恢复后的清理结果反馈

## 演示要点

- 强调“清理前解释、清理中反馈、清理后可恢复”
- 强调“重复文件”和“热点目录”两条不同治理路径
- 强调“任务”和“报告”帮助用户形成可回看记录
