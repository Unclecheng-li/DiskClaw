# 磁盘清理大虾 0.1.0 发布页文案

## 页面标题

磁盘清理大虾  
更安全的桌面端磁盘清理 AI Agent

## 页面副标题

帮你找到真正值得清理的空间黑洞，解释为什么能清、清了会怎样，并支持隔离回退。

## 核心卖点

### 1. 不只是扫描，更会解释

磁盘清理大虾不是传统的一键清理工具。  
它会结合本地扫描结果和 AI 分析，告诉你：

- 哪些内容最占空间
- 哪些内容更适合先清理
- 为什么推荐清理
- 清理后可能有什么影响

### 2. 重复文件与热点目录双线治理

除了常见的大文件、缓存、临时文件和安装包，本版还支持：

- 重复文件识别与保留建议
- 热点目录识别与治理建议

不只是“看见问题”，而是直接给出清理路径。

### 3. 清理可回退，更安心

默认优先采用隔离区模式。  
清理后的项目可以进入隔离区，并支持恢复最近一项或恢复选中项，减少误删焦虑。

### 4. 任务、报告、结果都可回看

从扫描到清理执行，整条链路都有任务反馈和报告记录：

- 任务进度
- 任务历史
- 报告筛选
- 报告详情

适合反复使用，也适合做演示与验证。

## 本版能力

- 本地磁盘扫描与空间分析
- 大文件、缓存、临时文件、日志、安装包候选识别
- 重复文件识别与保留建议
- 热点目录识别与治理建议
- 第三方 LLM API 接入
- AI 清理方案生成与问答解释
- 普通清理、重复文件清理、热点目录清理
- 隔离区恢复
- 任务进度、任务筛选与任务详情
- 报告筛选与报告详情
- Windows 安装包与桌面端工作台

## 适合谁

- 电脑空间紧张，但不确定该删什么的人
- 想清理下载目录、缓存目录、构建目录的用户
- 希望在执行前先看解释、执行后还能回退的人

## 当前版本说明

`0.1.0` 是磁盘清理大虾的桌面 MVP 版本。  
本版重点验证：

- 扫描分析
- AI 解释

- 任务与报告回看

当前仍以桌面单机使用与演示交付为主，默认不建议直接作为永久删除工具使用。
