# 磁盘清理大虾工作台 v2 GUI 最短验收路径

日期：2026-05-16  
适用版本：开发态 `npm run electron`

## 目标

用最短路径确认 v2 当前已做的 1~5 项是否基本可用：

- 危险操作确认
- 全局提示
- 日志中心
- 托盘与通知
- 窗口管理

## 步骤

### 1. 启动应用

执行：

```bash
npm run electron
```

### 2. 设置页验证

切到“设置”：

- 点击“读取桌面设置”
- 点击“测试系统通知”
- 点击“浏览文件夹”

如果三项都有响应，说明 Electron IPC 与桌面桥接基本正常。

### 3. 总览页验证

切到“总览”：

- 点击“填充示例配置”
- 点击“生成清理方案”
- 观察顶部任务状态条

如果状态条有变化，说明任务体系仍然正常。

### 4. 执行确认验证

切到“执行与报告”：

- 点击“执行普通清理”
- 确认弹出二次确认框
- 点击“取消”

如果弹框出现且取消后没有继续执行，说明危险操作确认已接通。

### 5. 日志中心验证

在页面底部系统日志区：

- 点击“展开全部”
- 输入关键词搜索
- 切换日志级别
- 点击“导出 JSON”

如果这些动作有响应，说明日志中心已经基本可用。

### 6. 窗口管理验证

- 调整窗口大小
- 关闭窗口
- 重新打开

如果窗口大小恢复，说明窗口状态记忆已生效。

## 通过标准

如果以上 6 步都正常，可以认为 v2 当前 1~5 项已经进入可继续打磨阶段，而不是不可用状态。
