## 一、商店短标题

## 二、商店一句话描述

更安全的桌面端磁盘清理 AI Agent，帮你找出空间黑洞、解释清理原因，并支持隔离回退。

## 三、商店短描述

磁盘清理大虾是一款桌面端磁盘清理 AI Agent。  
它不仅会扫描大文件、重复文件、缓存和热点目录，还会告诉你为什么建议清理、清理后会有什么影响，并支持隔离恢复，减少误删焦虑。

## 四、商店长描述

磁盘清理大虾面向需要整理磁盘空间的桌面用户，提供从“扫描分析”到“执行清理”再到“结果回看”的完整闭环。

本版支持：

- 本地磁盘扫描与空间分析
- 重复文件识别与保留建议
- 热点目录识别与治理建议
- 第三方 LLM API 接入与 AI 清理方案生成
- 普通清理、重复文件清理、热点目录清理
- 隔离区恢复
- 任务追踪与报告回看

如果你不只是想“一键删文件”，而是希望先理解、再执行、还能回退，磁盘清理大虾会更适合你。
