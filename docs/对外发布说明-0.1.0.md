# 磁盘清理大虾 0.1.0 对外发布说明

磁盘清理大虾（DiskClaw）是一款桌面端磁盘清理 AI Agent，帮助用户更安全地发现空间问题、理解清理建议，并完成可回退的清理执行。

## 本版亮点

- 支持本地磁盘扫描与空间分析
- 支持重复文件识别与保留建议
- 支持热点目录识别与治理建议
- 支持第三方 LLM API 接入与清理方案生成
- 支持普通清理、重复文件清理、热点目录清理
- 支持隔离区恢复与任务 / 报告回看
- 提供桌面端工作台、Windows 安装包与演示流程

## 适合谁

- 电脑空间紧张但不确定该删什么的普通用户
- 需要整理下载目录、缓存目录、构建目录的进阶用户
- 想先看解释、再做清理，且希望保留回退能力的用户

## 本版重点体验

- 首次启动引导
- 总览 / 设置 / 执行与报告 三分区工作台
- 首页摘要卡片、风险分布、优先事项一键动作
- 任务进度、任务历史、任务筛选
- 报告列表、报告筛选、报告摘要与详情

## 当前说明

`0.1.0` 为桌面 MVP 版本，重点验证“扫描分析 + LLM 解释 + 清理执行 + 隔离恢复 + 结果回看”的完整闭环。

目前默认采用隔离区模式，不建议直接作为永久删除工具使用。
