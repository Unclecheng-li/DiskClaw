# 磁盘清理大虾 开发态交互验收清单

日期：2026-05-16  
适用场景：`npm run electron`

## 一、启动前准备

1. 关闭所有已打开的 `DiskClaw` / `electron` 进程
2. 在项目根目录执行：

```bash
npm install
npm run electron
```

3. 等主窗口完全打开后再开始操作

## 二、基础页面交互

- [ ] 点击顶部 `总览 / 设置 / 执行与报告` 标签可以切换
- [ ] 点击“刷新状态”后顶部状态卡片会更新
- [ ] 点击“填充示例配置”后表单内容会变化

## 三、设置页交互

### LLM 配置

- [ ] 点击“读取已保存配置”有反馈
- [ ] 点击“恢复默认输入”后表单恢复默认状态
- [ ] 点击“测试连接”有提示结果
- [ ] 点击“保存配置”后有保存反馈

### 规则配置

- [ ] 点击“读取规则”后文本框内容可刷新
- [ ] 修改排除路径 / 白名单 / 黑名单后点击“保存规则”有反馈

### 偏好与周期清理

- [ ] 点击“读取偏好”后默认清理模式和偏好区域会更新
- [ ] 点击“保存偏好”后结果区会更新
- [ ] 点击“读取计划”后计划列表会刷新
- [ ] 填写计划表单后点击“保存计划”可新增计划
- [ ] 删除计划按钮可正常工作

## 四、总览页交互

- [ ] 点击“仅扫描”后有任务进度反馈
- [ ] 点击“生成清理方案”后有任务进度反馈
- [ ] 方案生成后摘要卡片更新
- [ ] 优先事项可点击并跳到对应操作
- [ ] Agent 问答可返回结果

## 五、执行与报告页交互

### 清理执行

- [ ] 普通清理 Dry Run 可执行
- [ ] 普通清理正式执行前会弹确认框
- [ ] 重复文件 Dry Run 可执行
- [ ] 热点目录 Dry Run 可执行
- [ ] `permanent` 模式下要求输入确认文字
- [ ] 清理结果统计卡片会更新

### 隔离区

- [ ] 点击“查看隔离区”后可看到列表
- [ ] 可勾选多个隔离项
- [ ] “恢复最近一项”可执行
- [ ] “恢复选中项”可执行

### 报告

- [ ] 点击“加载报告索引”后列表刷新
- [ ] 报告筛选按钮可正常工作
- [ ] 点击报告列表项可加载详情
- [ ] “导出最新报告 JSON / Markdown”可执行
- [ ] 近 7 / 30 天统计卡片可显示

### 任务

- [ ] 点击“加载任务列表”后列表刷新
- [ ] 任务筛选按钮可正常工作
- [ ] 点击任务列表项可加载详情
- [ ] 有活跃任务时“暂停 / 继续 / 取消”按钮可触发

- 实际结果
- 预期结果
- 是否可稳定复现
