## 500 字版

磁盘清理大虾（DiskClaw）是一款面向桌面端的磁盘清理 AI Agent，目标不是简单“一键删文件”，而是帮助用户更安全地完成从发现问题到执行清理的完整闭环。

本版支持本地磁盘扫描与空间分析，可识别大文件、缓存、临时文件、日志、安装包、重复文件和热点目录，并结合第三方 LLM API 生成可解释的清理方案。用户不仅能看到“哪些内容最占空间”，还能看到“为什么建议清理”“清理后可能产生什么影响”，降低误删焦虑。

为了提升安全感，磁盘清理大虾默认优先采用隔离区模式。清理后的项目可进入隔离区，支持恢复最近一项或恢复选中项。与此同时，系统还提供任务进度、任务历史、报告筛选、报告详情和执行回看，帮助用户在每次清理后保留清晰记录。

如果你需要一款更注重解释、回退能力和执行闭环的桌面端磁盘清理工具，磁盘清理大虾会是一种更稳妥的选择。
