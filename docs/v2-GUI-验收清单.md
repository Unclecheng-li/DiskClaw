# 磁盘清理大虾工作台 v2 GUI 验收清单

日期：2026-05-16  
适用版本：当前开发态 `npm run electron`

## 一、启动前准备

1. 关闭所有 `DiskClaw` / `electron` 进程
2. 在项目目录执行：

```bash
npm run electron
```

3. 等待主窗口完全打开

## 二、危险操作二次确认

### 普通清理

- [ ] 点击“执行普通清理”会弹出确认框
- [ ] 确认框显示影响范围（文件数、空间）
- [ ] 点击“取消”后不会执行清理
- [ ] 低风险操作可勾选“不再提示”

### 重复文件清理

- [ ] 点击“执行重复文件隔离”会弹出确认框
- [ ] 确认后才真正进入任务执行

### 热点目录清理

- [ ] 点击“执行热点目录隔离”会弹出确认框
- [ ] 取消后不会发起任务

### 永久删除模式

- [ ] 将执行模式切换到 `permanent`
- [ ] 点击任意执行按钮会要求输入确认文字
- [ ] 输入错误时不能继续

### 恢复操作

- [ ] 点击“恢复最近一项”会弹确认框
- [ ] 点击“恢复选中项”会弹确认框

## 三、全局提示系统

- [ ] 成功操作出现成功提示
- [ ] 取消操作出现警告提示
- [ ] 请求失败出现错误提示
- [ ] 错误提示带“查看详细日志”入口
- [ ] 点击“查看详细日志”后页面滚动到日志区

## 四、系统日志中心

- [ ] 默认只显示最近 5 条摘要
- [ ] 点击“展开全部”可查看更多日志
- [ ] 点击“收起”恢复摘要视图
- [ ] 搜索关键词可过滤日志
- [ ] 级别筛选可生效
- [ ] 时间筛选可生效
- [ ] 点击“导出 TXT”可导出日志
- [ ] 点击“导出 JSON”可导出日志
- [ ] 点击“清空日志”会先弹确认框
- [ ] 清空后日志区恢复空状态

## 五、托盘与通知

- [ ] 托盘图标存在
- [ ] 托盘菜单包含“打开主窗口 / 快速扫描 / 退出应用”
- [ ] 点击“测试系统通知”有系统通知
- [ ] 托盘点击后可重新显示主窗口

## 六、窗口管理

- [ ] 调整窗口大小后关闭重开，大小会恢复
- [ ] 移动窗口位置后关闭重开，位置会恢复
- [ ] 勾选“总是置顶”后窗口置顶生效
- [ ] 设置“关闭窗口时的行为”为“最小化到托盘”后，关闭按钮会隐藏到托盘
- [ ] 设置为“询问”后，关闭时会弹选择框

## 七、桌面端设置

- [ ] 点击“读取桌面设置”能刷新当前值
- [ ] 点击“保存桌面设置”有成功反馈
