# 磁盘清理大虾 0.1.1 修复版任务清单

版本目标：0.1.1  
日期：2026-05-15

## 版本定位

`0.1.1` 建议作为 `0.1.0` 桌面 MVP 的修复与打磨版本，重点处理稳定性、显示细节和交互一致性问题，不做大范围功能扩张。

## P0 必做

- [ ] 处理 Electron 相关脚本中的 `DEP0190` shell 警告，减少发布验证噪声
- [ ] 给任务详情和报告详情增加更统一的摘要字段兜底，避免部分记录只显示原始 JSON
- [ ] 对恢复流程补充更多异常提示，如“文件已不存在”“目标路径冲突”

## P1 建议做

- [ ] 给设置页增加“恢复默认值”后的视觉提示反馈强化
- [ ] 给任务列表和报告列表增加更清晰的状态 / 类型标签颜色
- [ ] 给隔离区多项恢复增加“全选 / 清空选择”
- [ ] 给总览页增加风险占比或轻量图形展示

## P1 稳定性补强

- [ ] 为报告筛选、任务筛选、多项恢复新增自动化测试
- [ ] 为 `verify:release` 增加对安装包产物存在性的更明确断言
- [ ] 为桌面烟雾检查补更稳的窗口检测策略

## P2 可选优化

- [ ] 将详情区中的原始 JSON 改成可折叠显示
- [ ] 给“关于磁盘清理大虾”窗口增加图标与更完整的版本说明
- [ ] 整理 README 中偏开发视角的内容，补更清晰的使用示例

## 建议节奏

### 阶段 1：稳定性

- 完成 P0
- 补充自动化验证

### 阶段 2：体验打磨

- 统一细节反馈
