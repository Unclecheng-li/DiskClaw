# 磁盘清理大虾 0.1.0 版本说明

日期：2026-05-14

## 版本定位

`0.1.0` 是磁盘清理大虾的桌面 MVP 版本，目标是验证“扫描分析 + LLM 解释 + 清理执行 + 结果回看”的完整工作闭环。

## 本版能力

- 桌面端 Electron 壳
- Windows 安装包构建
- 第三方 LLM API 接入
- 本地磁盘扫描
- 大文件、缓存、安装包、临时文件、日志候选识别
- 重复文件识别与保留建议
- 热点目录识别与治理建议
- 普通清理、重复文件清理、热点目录清理
- 隔离区恢复
- 任务进度、任务筛选与任务详情
- 报告筛选与报告详情

## 本版体验增强

- 首次启动引导
- 总览 / 设置 / 执行与报告 三分区工作台
- 首页方案摘要、风险分布、优先事项
- 优先事项一键触发对应 Dry Run
- 设置页支持恢复默认值

## 当前限制

- 永久删除仍未开放，当前以隔离区模式为主
- 详情区仍保留原始 JSON 视图，适合开发与验证
- 自动更新尚未接入
- 多项恢复目前仅支持隔离区，不支持更复杂的批量回滚策略

## 建议的下一版本方向

- 自动更新准备
- 更细的执行结果可视化
- 报告导出
- 更完整的设置体系
- 更丰富的清理规则库
